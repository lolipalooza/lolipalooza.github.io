Anotaciones

  • text-align: permite posicionar texto a la derecha/izquierda/centro/justificado.
  • text-decoration: trazar línea en texto (arriba/abajo/en medio).
  • text-transformation: convertir a mayúsculas/minúsculas/letras iniciales(capitalizar)
  • text-indent: añadir sangría
  • font-style: permite colocar letras en italic, o cursivas en español jeje ya se me olvidaba como se decía italic en español jejejejeejej
  • enlaces: a:link/visited/hover/active
  • Listas: list-style-type: circle/square(unorderer - ul) - upper-roman/lower-alpha(orderer - ol), list-style-image: url('');
  • vertical-align: top/bottom/middle.
  • border styles. Outline property.
  • The display property: inline/block/none/
  • display:none vs visibility:hidden - La segunda oculta el elemento pero deja el espacio donde éste se encontraba.
  • margin:auto - para que text-align:center pueda colocar en el centro no solo el interior de un block element (div, por ejemplo) sino también pueda colocar en el centro el block element entero (en caso de que tenga un width inferior al 100%). También se puede usar margin:auto combinado con width: 300px para centrar block element.
  • position: static/relative/fixed/absolute van de la mano con las propiedades left,right,top,botton properties. Nota: Left manda sobre Right y Top manda sobre Bottom. Se puede posicionar un block element con psoition:absolute y right:0px.
  • Float (left/right/none/inherit). Se puede usar con Clear (left/right/both/none/inherit).
  • display:inline-block se puede usar casi como un float:left. el inline-block se comporta como un elemento inline, pero a diferencia de éste último, se le pueden asignar propiedades width y height, ¿cómo la ves pendejete?
  • Selección múltiple de elementos:
    • Descendientes: div p {}
    • Descendientes directos (hijos/child): div > p {}
    • Elementos adyacentes (hermanos/sibling): div + p {} Nota: la nomenclatura div + p significa seleccionar el primer p que venga luego de un div.
    • Todos los elementos consiguientes (general siblings): div ~ p {} Nota: la nomenclatura div ~ p significa seleccionar todos los p que vengan luego de un div. Nota2: sorpresivamente, ésto elige todos los p luego del div sin importar que hayan otros elementos atravesados entre el div y los p siguientes!!!
  • Pseudo-clases: a:link, a:visited, a:hover, a:active. Otros ejemplos: p:first-child seleccionará al elemento p que es primer hijo de alguien. Mas ejemplos: :lang, :focus. Buscar referencia para ver MOAS.
  • Pseudo-elementos: ::first-letter, ::first-line, ::after, ::before, ::selection (para mozilla es ::-moz-selection) jeje es ::neta.
  • La propiedad opacity para imágenes (valores de 0.0 a 1.0). También se puede usar filter: alpha(opacity=x) donde x puede variar de 0 a 100. También se puede usar opacity en divs con background colors.
  • Truco para mostrar imágenes/botones a partir de un sprite/sheet aqui.
  • Tipos de @media: ver aqui.
  • Seleccionar elementos con atributos: a[atribute], a[attribute="value"], etc. Ver más en este capítulillo jeje.
